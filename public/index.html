< About Me />

I'm Jose A. Echezuria a sysadmin and homelab enthusiast from Venezuela, now based in Houston, TX. I build and manage self-hosted infrastructure, specializing in Linux systems, Docker containerization, and DevOps practices. My homelab runs on Proxmox with services like PostgreSQL, Nginx, Pi-hole, Grafana, and custom applications all deployed with Docker and monitored 24/7. I'm self-taught through hands-on homelabbing, bootcamps, and certifications, and I'm about to start a Computer Science degree to formalize my knowledge. From automating deployments to optimizing server performance, if it's open-source, self-hosted, and efficient... I'm into it.
